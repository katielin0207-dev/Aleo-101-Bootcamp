### `abs`

```leo
let a: i8 = -1i8;
let b: i8 = a.abs(); // 1i8
```

### `div`

```leo
let a: u8 = 4u8;
let b: u8 = a / 2u8; // 2u8
let c: u8 = b.div(2u8); // 1u8
```

将第一操作数除以第二操作数，结果存储在目标位置。如果尝试除以零，操作将停止。

对于整数类型，此操作执行截断除法——向零取整，无论操作数的符号如何，即截掉小数部分，保留绝对值不超过结果的整数。

例如：

- `7 / 3` 的结果是 `2`，而不是 `2.3333`。
- `-7 / 3` 的结果是 `-2`，而不是 `-2.3333`。

如果发生下溢，操作将停止。下溢发生在有符号整数类型的最小值除以 -1 时。例如，`-128i8 / -1i8` 会下溢，因为 `128` 无法用 `i8` 表示。

对于域类型，除法 `a / b` 在 `b = 0field` 时未定义，其他情况均定义。

对于需要整数回绕语义的情况，请参阅 [div_wrapped](#div_wrapped)。

#### 支持的类型

| 第一操作数 | 第二操作数 | 目标类型    |
| ------- | ------- | :---------- |
| `field` | `field` | `field`     |
| `i8`    | `i8`    | `i8`        |
| `i16`   | `i16`   | `i16`       |
| `i32`   | `i32`   | `i32`       |
| `i64`   | `i64`   | `i64`       |
| `i128`  | `i128`  | `i128`      |
| `u8`    | `u8`    | `u8`        |
| `u16`   | `u16`   | `u16`       |
| `u32`   | `u32`   | `u32`       |

### `mod`

```leo
let a: u8 = 3u8.mod(2u8); // 1u8
```

计算 `first` 对 `second` 取模的结果，存储在 `destination` 中。如果 `second` 为零则停止。

此操作的语义与数学上取模运算的定义一致。

`mod` 保证余数与 `second` 操作数同号，这与 [`rem`](#rem) 不同——后者遵循截断除法，余数与 `first` 操作数同号。

| `u8`   | `u8`   | `u8`        |

### `rem`

```leo
let a: u8 = 3u8 % 2u8; // 1u8
let b: u8 = 4u8.rem(2u8); // 0u8
```

按照截断除法规则计算 `first` 除以 `second` 的余数，存储在 `destination` 中：

其中 a 和 b 分别指 first 和 second

`a % b = a - (a / b) * b`

注意，如果运算下溢或除以零，执行将停止。下溢发生在相关除法操作 [div](#div) 下溢时。

对于整数类型需要回绕语义的情况，请参阅 [rem_wrapped](#rem_wrapped)。

`rem` 遵循截断除法，余数与 `a` 同号，这与 [mod](#mod) 不同——后者余数与 `b` 同号。

| 第一操作数 | 第二操作数 | 目标类型    |
| ------ | ------ | ----------- |
| `i8`   | `i8`   | `i8`        |
| `i16`  | `i16`  | `i16`       |
| `i32`  | `i32`  | `i32`       |
| `i64`  | `i64`  | `i64`       |
| `i128` | `i128` | `i128`      |
| `u8`   | `u8`   | `u8`        |
| `u16`  | `u16`  | `u16`       |
