# 运算符与表达式

Leo 中的运算符基于一个或多个表达式计算出一个值。Leo 默认使用检查算术，即当检测到溢出或除以零时会抛出错误。

以加法为例：`first` 与 `second` 相加，结果存储在 `destination` 中。对于整数类型，会添加约束来检查溢出。
对于需要回绕语义的整数类型情况，请参阅运算符的带 wrapped 后缀变体。

```leo
let a: u8 = 1u8 + 1u8;
// a 等于 2

a += 1u8;
// a 现在等于 3

a = a.add(1u8);
// a 现在等于 4
```

:::note
Leo 运算符会编译为可由 Aleo 虚拟机（AVM）执行的 [`Aleo Instructions`](https://developer.aleo.org/guides/aleo/opcodes) 操作码。
:::

## 运算符优先级

运算符按以下顺序优先求值：

|                                   运算符                                    | 结合性 |
| :---------------------------------------------------------------------------: | :-----------: |
|                                `!` `-`（一元）                                 |               |
|                                     `**`                                      | 从右到左 |
|                                    `*` `/`                                    | 从左到右 |
|                                `+` `-`（二元）                                | 从左到右 |
|                                   `<<` `>>`                                   | 从左到右 |
|                                      `&`                                      | 从左到右 |
|                              <code>&#124;</code>                              | 从左到右 |
|                                      `^`                                      | 从左到右 |
|                               `<` `>` `<=` `>=`                               |               |
|                                   `==` `!=`                                   | 从左到右 |
|                                     `&&`                                      | 从左到右 |
|                           <code>&#124;&#124;</code>                           | 从左到右 |
| `=` `+=` `-=` `*=` `/=` `%=` `**=` `<<=` `>>=` `&=` <code>&#124;=</code> `^=` |               |

## 括号

若需要改变求值顺序，可使用括号 `()` 将表达式括起来。

```leo
let result = (a + 1u8) * 2u8;
```

`(a + 1u8)` 会在乘以 `* 2u8` 之前先被求值。
